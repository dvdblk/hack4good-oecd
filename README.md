## Conda Environment

> Requirements: conda

To **create** a conda environment after cloning the repo:
```
conda create -n hack4good --file conda-req.txt
```

To **activate** this environment:
```
conda activate hack4good
```

**Deactivate** this environment with:
```
conda deactivate
```
